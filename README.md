# NEON DEFENSE

An 8-bit neon tower-defense game, built as a **single self-contained HTML file** — no build step,
no dependencies, no asset files. Every graphic is drawn on an HTML5 `<canvas>` and every sound is
synthesized with the Web Audio API.

**▶ Play:** https://shoemocker.github.io/neon-defense/  ·  or just open `index.html` in any browser.

## Features
- 7 towers (Cannon, Gatling, Frost, Venom, Tesla, Psyche, Demolition) with coin upgrades that grow them
- Colour-coded Pac-Man ghosts with per-colour tower resistances
- A boss every 5th wave — pick one of three permanent buffs, then kill it to claim
- Special enemies that **destroy / freeze / stun / weaken** your towers
- **Procedurally generated paths that route around your towers** — different every game
- 3 abilities (Overdrive · Phantom Hunt · Reveal Path), a speed control + boss auto-ramp,
  a Boss-Credit currency for premium towers, a live DPS board, and a high-score table

## Run it locally
Open `index.html` in any modern browser — it's completely self-contained.
